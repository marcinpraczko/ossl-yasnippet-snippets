Documentation for this project
==============================

This documentaiton is written in RST format and compiled in sphinx-doc. You
can read this documentation format in RST format (as is) or install
sphinx-doc, navigate to docs folders and run:

* On Linux systems

.. code-block:: bash

   make html
   make text

* On Windows systems 

.. code-block:: bash

   make.bat html
   make.bat text

This will generate documentation in HTML and TEXT format in folder:

* For html: **docs/build/html**
* For text: **docs/build/text** (Included to github)
